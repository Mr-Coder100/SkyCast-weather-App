# SkyCast - Weather App 🌤️

A clean, modern weather app built with Flask and OpenWeatherMap API.

## Features
- 🌡️ Current temperature + feels like
- 💧 Humidity
- 💨 Wind speed
- 🔵 Atmospheric pressure
- ☁️ Cloud coverage
- 👁️ Visibility
- 🌅 Sunrise & Sunset times
- 📅 5-Day forecast

## Setup

1. Clone the repo
2. Install dependencies:
   ```
   pip install -r requirements.txt
   ```
3. Add your OpenWeatherMap API key in `config/config.cfg`
4. Run:
   ```
   python app.py
   ```
5. Visit `http://localhost:5000`

## API
Uses [OpenWeatherMap](https://openweathermap.org/) free tier API.
